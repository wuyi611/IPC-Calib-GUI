# IPC-Calib-GUI

**IPC-Calib-GUI** 是一款专为网络相机设计的图形化标定工具，集成 **ONVIF 自动发现**、**多线程 RTSP 流处理**与**双窗口实时畸变矫正对比**功能。

---

## 🌟 项目亮点

* **IPC 智能发现**：集成 ONVIF 协议，一键扫描局域网内的网络相机，并自动识别主、次码流地址。
* **多线程架构**：采用独立线程负责 RTSP 拉流与标定计算，彻底解决连接相机及运行图像处理算法时的界面卡顿问题。
* **可视化对比预览**：提供纵向双窗口模式，实时标注 “Original”（原图）与 “Rectified”（矫正后）画面，方便直观评估标定效果。
* **集成日志系统**：在 GUI 左下角内置实时日志窗口，重定向标准输出与错误信息，动态掌握扫描、截图及计算进度。
